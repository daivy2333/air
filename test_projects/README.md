# 测试项目集合

这个目录包含用于测试解析器的各种语言项目。

## 项目结构

- python_project/ - Python 测试项目
- c_project/ - C 测试项目
- cpp_project/ - C++ 测试项目
- rust_project/ - Rust 测试项目
- asm_project/ - 汇编测试项目
- cross_project/ - 跨语言混合项目
- os_project/ - 简单的操作系统内核项目

## 使用方法

每个项目都包含：
1. 源代码文件
2. 对应的 PIR 文件
3. 测试用的 PCR 文件
4. README 说明文档
